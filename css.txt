Cuando aumenta el (margin) margen del cuadro azul, aumentará la distancia entre su borde y los elementos circundantes.

-----------Colores-------------------
El dígito 0 es el número más bajo en código hexadecimal y representa una ausencia completa de color.

El dígito F es el número más alto en código hexadecimal y representa el brillo máximo posible.


Dodger Blue	#1E90FF
Green	#00FF00
Orange	#FFA500
Red	#FF0000

Por ejemplo, el código hexadecimal rojo # FF0000 se puede acortar a # F00. Esta forma abreviada da un dígito para el rojo, un dígito para el verde y un dígito para el azul.

Esto reduce la cantidad total de colores posibles a alrededor de 4,000. Pero los navegadores interpretarán # FF0000 y # F00 exactamente del mismo color.

Cyan	#0FF
Green	#0F0
Red	#F00
Fuchsia	#F0F

Sienna	rgb(160, 82, 45)

---------------Variables en css----
Para crear una variable CSS, solo necesita darle un nombre con dos guiones delante y asignarle un valor como este:

    --penguin-skin: gris;

Esto creará una variable llamada --penguin-skin y le asignará el valor de gris.

Ahora puede usar esa variable en cualquier lugar de su CSS para cambiar el valor de otros elementos a gris.

.penguin {
    
     
 --penguin-skin: gray;
}

//Usando la variable

.penguin-top {

	background:var (--penguin-skin);

  }

En caso de que el navegador no sea compatible para crear las variables con css
background: var(--penguin-skin, black);

Scope de las varibles en css:
Cuando crea una variable, está disponible para que la use dentro del elemento en el que la creó. También se vuelve disponible dentro de cualquier elemento anidado dentro de él. Este efecto se conoce como cascada.

Debido a la cascada, las variables de CSS a menudo se definen en el elemento raíz (:root).

Puede pensar en: el elemento raíz como un contenedor para todo su documento HTML, de la misma manera que un elemento html es un contenedor para el elemento del cuerpo.

Al crear sus variables en: root, estarán disponibles en toda la página web.

:root {
 
	--penguin-belly:pink;
}
